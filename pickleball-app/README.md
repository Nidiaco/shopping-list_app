# Pickle Champs 🥒

A mobile-first web app for tracking weekly pickleball sessions and player performance. Built for a small group in Christchurch, New Zealand.

## What it does

Players log in to view upcoming sessions, check the leaderboard, and follow live scores. An admin account manages sessions, records game results, and maintains the player roster.

## Features

- Session scheduling with Google Maps integration
- Live score recording with team rotation (2v2)
- Player leaderboard with win rate and point differential
- Role-based access — view for players, full control for admin
- Real-time sync across all devices

## Tech

HTML, CSS, JavaScript, Firebase Firestore
